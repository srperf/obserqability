# Tested Software

These are the examples of tested software we will be using.

The first one is a web page rich in buttons, fields and elements that a functional browser automation can play with.

The second is a simple web service that will receive some elements and respond. Plain and simple.

## Starting the software

To start the software you can rrun them using node:

```
node service.js
```

```
node webpage.js
```

Both can be accessed using just a web browser through localhost and their respective port (4000 for the webpage and 4001 for the web service).
